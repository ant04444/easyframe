== Installation

=== Prerequisites

* Java: 1.8

=== Option 1: Runs with docker
* Ensure the easyframe network is available
----
docker network create easyframe
----
* Run the applicaiton with docker-compose
----
docker-compose up -d
----

=== Options 2: Runs with standalone java

----
mvn spring-boot:run
----
